## :clipboard: Tech Stack
- Minimum SDK level 26(Android 8.0)
- Programming language: Kotlin
- Architecture: Clean + MVVM pattern
- Dependency Injection: Hilt
- Async: Coroutines + Flow
- DataBase: Room
