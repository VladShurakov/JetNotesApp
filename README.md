Notes app project on Android written in Kotlin with Jetpack Compose

Feel free to make issues, forks and pull requests

## :clipboard: Tech Stack
- Minimum SDK level 26
- Clean Architecture
- MVVM pattern
- DI: Hilt
- Async: Coroutines
- DataBase: Room
